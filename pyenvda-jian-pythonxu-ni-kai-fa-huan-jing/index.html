使用pyenv和virtualenv搭建python虚拟开发环境

pyenv install --list
查看有哪些版本python可以安装

pyenv install 3.6.2
安装3.6.2

pyenv versions
查看当前环境

pyenv virtualenv 3.6.2 env362
创建虚拟环境

pyenv activate env362
切换到环境

pyenv deactivate
切回原环境

rm -rf ~/.pyenv/versions/env362/
删除环境
