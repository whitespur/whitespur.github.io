接触的几套博客系统

1前言

目前主要接触了3套博客系统（wordpress，Goblog，ghost），分别基于php，go和nodejs开发的博客，wordpress用过一段时间，操作简便，不过后台管理不好用，摘要设置不好；goblog是基于beego框架开发的博客系统，数据存储mongodb，可docker部署，还整合了elasticsearch，项目地址（https://github.com/deepzz0/goblog）， 用的技术都比较新潮，在本地环境mac上搭起来了，但是在服务器上面一直没部署起来，主要由于我买的服务器提供的linux内核都是2.6.32的版本，而docker要求内核至少3.10，也懒得升级内核，就一直没弄了；最近看到了ghost的博客，外观简洁清晰，就了解了下，今天花了点时间搭起来了。

2 相关环境

centos6.6 64位系统,nodejs,nginx,mysql

3 具体步骤

3.1 安装Ghost博客系统

http://docs.ghostchina.com/zh/installation/deploy/
https://docs.ghost.org/docs/install
https://segmentfault.com/a/1190000002947497
Forever (https://npmjs.org/package/forever)
你可以使用 forever 以后台任务运行 Ghost 。forever 将会按照 Ghost 的配置，当进程 crash 后重启 Ghost。 • 通过 npm install forever -g 安装 forever • 为了让 forever 从 Ghost 安装目录运行，输入 NODE_ENV=production forever start index.js • 通过 forever stop index.js 停止 Ghost • 通过 forever list 检查 Ghost 当前是否正在运行
