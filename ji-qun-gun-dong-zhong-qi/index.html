Elasticsearch集群滚动重启

参考文章 https://elasticsearch.cn/book/elasticsearchdefinitiveguide2.x/rolling_restarts.html
http://blog.csdn.net/hereiskxm/article/details/45717573

1关闭自动分片

curl -XPUT http://username:passwd@192.168.181.196:9200/cluster/settings -d '
{ "transient" : { "cluster.routing.allocation.enable" : "none" } }'

2 修改配置，重启节点：

/es_home/bin/service/elasticsearch.conf

/es_home/config/elasticsearch.yml

elasticsearch restart

3 开启自动分片

curl -XPUT http://username:passwd@192.168.181.196:9200/cluster/settings -d '
{ "transient" : { "cluster.routing.allocation.enable" : "all" } }'
